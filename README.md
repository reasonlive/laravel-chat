### Laravel chat
#### REST API on Laravel 12 with Octane + Swoole
#### Client app on React with Typescript + Antd

##### Download repository:
```bash
git clone https://github.com/reasonlive/laravel-chat.git
cd laravel-chat
```
##### Start application:
```bash
docker-compose build
docker-compose up
```
##### Stop application:
```bash
docker-compose down
```

##### Application run on http://localhost:3000
